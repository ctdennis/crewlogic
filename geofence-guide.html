CrewLogic · Dispatch

Geofence Setup Guide

CrewLogic reads your telematics geofences to show, on the dispatch map, which truck is at which site, how long it's been there, and a flag when it's been there too long. Getting that working comes down to labeling each geofence — a Category in Motive, a Group in Linxup.

The label drives it — not the geofence's name.  In Motive you set a geofence's Category; in Linxup you put the geofence in a Group. Use the same words either way — CrewLogic reads both.

02

Motive — set the Category

  1. Open Fleet → Geofences and click the geofence.
  2. Set its Category to the matching one above, then Save.
  3. Make sure the geofence is Active.
  • →
    Leave a site Uncategorized and it gets no badge — so categorize every transfer, recycling, and donation site you use.
  • →
    Non-facility geofences get no badge on purpose — tag your home base Headquarters and service areas (Cape Cod, Rhode Island, etc.) Region to keep them labeled.
03

Linxup — set the Group + the webhook

Linxup pushes events to CrewLogic, so there are two parts: put each geofence in the right Group, and connect the webhook once.

  1. Put each geofence in a Group named Transfer Station, Recycling, or Donations.
  2. In CrewLogic → Settings → Trucks, copy the webhook URL shown (it already includes your account) and click Generate token.
  3. In Linxup's Push API config: paste that URL into Geofence Event URL, and paste CrewLogic's generated token into Bearer Token.
⚠️
The Bearer Token must be the token CrewLogic generated — not your Linxup API key. CrewLogic makes the token; you paste it into Linxup. (Your Linxup API key goes in the separate "Linxup REST token" box in CrewLogic, which pulls truck positions.)
04

Set your wait-times

The red “too long” flag fires when a truck has been at a site past your wait-time for that type.

⏱
CrewLogic → Settings → Wait Times (minutes per load) — one number each for Disposal, Recycling, and Donation.
05

Quick checklist

Every transfer/disposal, recycling, and donation site has a geofence
Each has the right Category (Motive) or Group (Linxup) — not left blank
Each geofence is Active
Linxup only: webhook URL + CrewLogic-generated token pasted into Linxup
Wait Times set in CrewLogic Settings
